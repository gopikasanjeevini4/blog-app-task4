# Blog App - ApexPlanet Internship

A web-based Blog Application built with PHP and MySQL as part of the 45-day Web Development Internship at ApexPlanet Software Pvt. Ltd.

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- XAMPP (Apache + MySQL)

## Features
- User Registration and Login
- Create Blog Posts
- Read/View All Blog Posts
- Edit Blog Posts
- Delete Blog Posts
- Session Management
- Password Hashing for Security

## Database
- Database Name: blog
- Tables: users, posts

## How to Run
1. Install XAMPP
2. Start Apache and MySQL
3. Clone this repository into C:\xampp\htdocs\blog
4. Open phpMyAdmin and create database named blog
5. Create users and posts tables
6. Open browser and go to http://localhost/blog
